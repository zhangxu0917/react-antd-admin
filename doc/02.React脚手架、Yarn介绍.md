## React脚手架、Yarn介绍

### 如何安装和使用React脚手架
```bash
npm i -g create-react-app
create-react-app my-app

cd my-app
npm start
```

### 什么是Yarn
- Yarn是新一代的包管理工具
### 为什么使用Yarn
- 速度快
- 安装版本统一、更安全
- 更简洁的输出
- 更好的语义化
### 如何使用Yarn
- `yarn init`
- `yarn add`
- `yarn remove`
- `yarn / yarn install`
